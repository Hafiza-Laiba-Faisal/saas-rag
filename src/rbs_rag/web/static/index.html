☁️ Sync Cloud Storage Documents
Download files directly from Google Drive, OneDrive, or Direct URLs directly into this client's isolated workspace.
💡 How to get a Google Drive Folder / File ID:
  1. Open your folder or file in Google Drive.
  2. Look at your browser URL bar: https://drive.google.com/drive/folders/YOUR_FOLDER_ID
  3. Copy the string after /folders/ or /d/ and paste it here.
  4. Ensure link sharing is set to "Anyone with the link can view".
💡 How to create a Google API Key:
  1. Go to Google Cloud Console (console.cloud.google.com).
  2. Navigate to APIs & Services > Credentials.
  3. Click Create Credentials > API Key.
  4. Enable the Google Drive API under Enabled APIs.
💡 How to get a OneDrive Share Link:
  1. Right-click your file/folder in OneDrive web.
  2. Click Share and set permission to "Anyone with the link can view".
  3. Copy the link and paste it here.
💡 Direct File Link Guide:

Paste any direct URL ending in .pdf, .docx, .md, .txt, or .html. The file will be fetched securely into this tenant's isolated directory.
